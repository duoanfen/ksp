Redis安装

Redis安装

下载软件包, 登录Linux:

cd /usr/local/

wget http://download.redis.io/releases/redis-3.2.2.tar.gz

tar -zxvf redis-3.0.3.tar.gz

mv redis-3.0.3  redis

cd redis

make && make install
